## Experiments

### exp-01: OpenCV Installation and Basic Image Handling
- Load, display, resize, and save images
- Convert images to grayscale

---

### exp-02: Edge, Line, and Corner Detection
- Edge detection using Canny
- Line detection using Hough Transform
- Corner detection using Harris method

---

### exp-03: Camera Calibration
- Detect chessboard corners
- Compute camera matrix and distortion coefficients
- Undistort images

---

### exp-04: Histogram and Histogram Equalization
- Plot image histogram
- Improve contrast using histogram equalization

---

### exp-05: Skin Color Detection
- Detect skin regions using HSV color space
- Apply masking and noise removal

---

### exp-06: Image Warping and Perspective Transformation
- Perform perspective transform
- Convert image viewpoint (e.g., top view)

---

### exp-07: Motion Tracking
- Detect moving objects using frame differencing
- Draw bounding boxes around motion

---

### exp-08: Object Detection using YOLO
- Load YOLO model
- Detect and label objects in images

---

### exp-09: Stereo Vision and Depth Estimation
- Compute disparity map using stereo images
- Estimate depth information

---

### exp-10: Augmented Reality using Feature Matching
- Detect features using ORB
- Match features and compute homography
- Overlay virtual image on real scene
